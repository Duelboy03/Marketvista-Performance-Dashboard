# Marketvista-Performance-Dashboard
Interactive Power BI dashboard analyzing revenue, profit, targets, and regional performance for MarketVista
📊 MarketVista Performance Dashboard (Power BI)

📌 Overview

Developed an interactive Power BI dashboard to analyze business performance for MarketVista. The dashboard provides insights into revenue, profitability, return rates, and store-level performance across regions.

🛠 Tools & Technologies
	•	Power BI
	•	Power Query (ETL & Data Transformation)
	•	DAX (Data Analysis Expressions)

📊 Key Features
	•	KPI cards displaying Total Revenue ($1.8M), Profit ($1.1M), Profit Margin (59.67%), Return Rate (0.99%), and Revenue Target ($1.7M)
	•	Revenue vs Target comparison to evaluate performance gaps
	•	Monthly revenue trend analysis
	•	Store-level performance visualized using map
	•	Brand-level performance analysis

💡 Key Insights
	•	Revenue exceeds target, indicating strong overall performance
	•	Certain brands outperform others in terms of profitability
	•	Regional distribution highlights key revenue-generating areas
	•	Stable upward trend observed in monthly revenue
